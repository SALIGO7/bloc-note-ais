Formules à retenir

Magic Number
  • Hôtes = 2^n − 2   (n = bits hôtes)
  • Block size = 256 − valeur octet masque
  • Ex: /26 → 256 − 192 = 64
  • Sous-réseaux = 2^(bits empruntés)
  • Réseau suivant = réseau actuel + block size
Binaire — les 8 puissances
1286432168421

192 = 128+64 → 11000000 → /26

255.255.255.224 → 32ème bit hôte → /27

Active Directory — Windows Server

Domaine

Unité admin AD. Ex: hydrogel.local. Contient users, groupes, OUs, GPOs, ordinateurs.

OU

Conteneur logique pour GPO ciblées. Peut être imbriquée. Ex: OU=Compta,DC=hydrogel,DC=local

DC

Serveur AD DS. Auth Kerberos. Min. 2 DC recommandés pour redondance.

Catalogue Global

Copie partielle de tous les objets de la forêt. Nécessaire pour auth inter-domaines.

Forêt

Ensemble de domaines liés. Frontière de sécurité ultime en AD.

FSMO (5 rôles)

Modèle AGDLP

Account → Global Group → Domain Local Group → Permission
  • A = Account (utilisateur ou ordinateur)
  • G = Global Group — regroupe par fonction (ex: GG_Compta)
  • DL = Domain Local Group — définit l'accès à une ressource (ex: DL_Lecture_Partage_Compta)
  • P = Permission NTFS appliquée au groupe DL sur la ressource
jdupont ──→ GG_Comptabilite ──→ DL_Acces_Partage_Compta ──→ Lecture sur \\SRV01\Compta Avantage : pour donner accès à un user, on l'ajoute au Global Group. Pas besoin de toucher aux permissions NTFS à chaque fois.

DFS — Namespace + Réplication

DFS Namespace (DFS-N)

Espace de noms unifié : \\domaine\partages. Masque l'emplacement physique. Facilite les migrations.

DFS Replication (DFS-R)

Réplique les dossiers entre serveurs. Algorithme RDC (delta). Remplace FRS. Topologie Hub-Spoke ou Mesh.

Diagnostics réseau — méthode & outils

Méthodologie bottom-up

1 — Physique

Câble branché ? LED switch allumée ? ping 127.0.0.1 répond ?

2 — Liaison

MAC visible ? Port switch UP ? ARP montre la passerelle ?

3 — Réseau

IP configurée ? Bon masque ? ping passerelle répond ?

4 — Transport

Port ouvert ? Firewall bloquant ? telnet IP port

5 — Application

Service démarré ? DNS résout ? Logs = erreurs ?

Matériel, Méthode, Milieu, Main d'œuvre, Matière — identifier la cause racine.
